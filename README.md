# Study Planner Pro

Aplicação web para organização de estudos.

## Funcionalidades

- Adicionar tarefas
- Marcar tarefas como concluídas
- Excluir tarefas
- Filtrar tarefas
- Estatísticas em tempo real
- Tema claro e escuro
- Salvamento automático com LocalStorage

## Tecnologias

- HTML5
- CSS3
- JavaScript
